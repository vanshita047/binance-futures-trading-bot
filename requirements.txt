# Binance Futures Trading Bot (Testnet)

## Overview
This project is a Python-based Binance Futures Trading Bot that allows users to place MARKET and LIMIT orders on Binance Futures Testnet via a command-line interface (CLI).

The project demonstrates:
- Binance Futures API integration
- Clean project structure
- Input validation
- Logging and error handling
- Testnet order execution

---

## Features

- Place MARKET orders
- Place LIMIT orders
- Input validation
- Structured logging (logs stored in logs/bot.log)
- Clean modular architecture

---

## Assumptions

- Orders are placed only on Binance Futures Testnet.
- Only MARKET and LIMIT orders are supported.
- LIMIT orders use `GTC` (Good Till Cancelled).
- User provides valid symbol and sufficient margin in testnet account.
